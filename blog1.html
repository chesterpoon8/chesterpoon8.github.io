Simple Linear Regression in Python¶

In most of the courses at CUNY SPS, including this one, R seems to be the preferred language to use when creating models and doing analysis. Since my first love is Python, this blog will just demonstrate how to create a simple linear regression model using Ski-Kit Learn, Seaborn, and Pandas

First, we'll import our libraries for our model and analysis

Visualizing our model using Seaborn and Scipy¶

The great thing about Seaborn is that there is a built in function to graph a simple linear regression model over our data. This is accomplished below. Let's first store our data into variables.

Loading the data into variables¶

Sci-kit learn's datasets are stored as numpy arrays. From the overview we saw earlier, we know that the 6th column in our array is the number of rooms. So we can access that data and store that in x. Median home value with be stored as y.

In [3]:
x = b.data[:,5]
y = b.target

Simple enough. Let's plot our data

Creating our model with sklearn¶

Now that we have a nice preliminary visual of our model through seaborn, let's build it with sklearn. We'll split our data 80/20, train/split. There are some limits to getting some statistical values using sklearn. In the event that one wants to get statistical values, a combination of scipy as demonstrated above and sklearn would work as well. Our statistical values here might also be slightly different since we'll be creating a training and test set.

In [6]:
model = LinearRegression()
X_train, X_test, y_train, y_test = train_test_split(x.reshape((-1,1)), #Number of Rooms
                                                    y, #Median Value of home
                                                    test_size=0.2,
                                                    random_state=42)

model.fit(X_train,y_train)
r2 = model.score(X_test,y_test)
intercept = model.intercept_
slope = model.coef_
y_pred = model.predict(X_test)
print('Slope: ',slope)
print('Intercept: ',intercept)
print('R-squared: ',r2)
